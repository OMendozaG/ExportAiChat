Provider Notes (Known Gaps)

ChatGPT: integrated with dedicated selectors for messages, title/folder, model, attachments, and references.

Claude: message, title, model and share-anchor extraction are integrated. Chat folder extraction is not identified from current logs.

Gemini: message, title, model, share-anchor and user upload extraction are integrated. Stable per-message timestamp hooks were not identified in current logs.

DeepSeek: message and visible thinking extraction are integrated. Header share-anchor/model selectors are still heuristic because the supplied logs rely on hashed class names.

Grok: message, visible thinking, file chip extraction and share-anchor extraction are integrated. Stable mode/model selectors were not confirmed in the supplied logs.
